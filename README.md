# 5tars v2 🌟

منصة SaaS عربية لإدارة تقييمات Google بالذكاء الاصطناعي

## 🚀 الإعداد السريع

1. افتح [setup_db.html](https://your-domain.vercel.app/setup_db.html) بعد النشر
2. أدخل `service_role` key من Supabase Dashboard
3. اضغط "تشغيل الإعداد" ← يُنشئ كل الجداول تلقائياً

## 📁 الملفات

| الملف | الوصف |
|-------|-------|
| `dashboard.html` | لوحة تحكم المستخدم |
| `finance.html` | لوحة المال (مالك فقط) |
| `review.html` | صفحة العميل الذكية |
| `login.html` | تسجيل الدخول |
| `signup.html` | إنشاء حساب |
| `setup_db.html` | إعداد قاعدة البيانات (مرة واحدة) |
| `database.sql` | SQL للإعداد اليدوي |
